# Python Crash Course (Eric Matthes): Self Study Sprint.  
Welcome to my Structured Repository for mastering Python from the first principles, This space acts a tracking log, codebase and knowledge base as I work through the text.  

## Repository Architecture  
Organised by Chapters. Each chapter directory follows a uniform three-tier structure.  
```text
📁 Python-Crash-Course/
│
├── 📁 Chapter-XX-[Chapter-Name]/
│   ├── 📁 sandbox/             # Active code-along files, experiments, and inline book examples.
│   ├── 📁 practice-problems/   # End-of-chapter exercises and challenges.
│   └── 📁 resources-used/           # Book annotations, cheat sheets, and reference material.
│
└── 📄 README.md                # Central hub, architecture overview, and core concepts log, with handwritten work attached if needed.
```
